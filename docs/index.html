Everything you need. Nothing you don't.

Zero Config

Point your OTLP exporter at port 4318 (HTTP) or 4317 (gRPC). SQLite database created automatically.

OTLP Native

Speaks OpenTelemetry natively over gRPC and HTTP. Compatible with every OTel SDK out of the box.

LLM Cost Tracking

Automatically detects LLM spans, extracts model and token counts, and calculates per-request costs.

SQLite Default

No database setup required. SQLite for development and small teams. Switch to PostgreSQL with one config line.

Built-in UI

Trace explorer, span waterfall, LLM observation detail, and API key management — served from the same binary.

API Key Auth

Optional Bearer token authentication for multi-tenant or production deployments. Managed via the Settings UI.

Quick Start

01

Pull & run

docker run -p 3000:3000 -p 4317:4317 -p 4318:4318 ghcr.io/raysca/litetrace:latest
02

Configure your exporter

OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4318 node your-app.js
03

Open the UI

open http://localhost:3000
