Quick Start

01

Pull & run

docker run -p 3000:3000 -p 4317:4317 -p 4318:4318 ghcr.io/raysca/litetrace:latest
02

Point your OTLP exporter at localhost:4318

OTEL_EXPORTER_OTLP_ENDPOINT=http://localhost:4318 node your-app.js

Works with any OpenTelemetry SDK — Python, Node.js, Go, Java, .NET, or any framework with OTel support.

03

Open the UI

open http://localhost:3000
